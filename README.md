# Ping Monitor

A real-time network monitoring system built using Prometheus, Grafana, and Blackbox Exporter.

## 🚀 Features
- Monitors latency, packet loss, and jitter across multiple endpoints
- Covers ISP services, local network devices, and game servers
- Uses ICMP and HTTP probing for availability and performance tracking
- Interactive dashboards for network health visualization and analysis
- Publicly accessible via web for external monitoring

## 🛠️ Tech Stack
- Prometheus
- Grafana
- Blackbox Exporter
- Cloudflare Tunnel
- GitHub Pages

## 🌐 Live Demo
https://sparxiconofc.github.io/pingmonitor/

## ⚠️ Notes
The dashboard is hosted on a limited runtime environment, so availability is not guaranteed at all times.

## 📊 Overview
This project was developed to analyze network performance trends and evaluate overall stability and reliability in real-time environments.
